# resume-parser
AI Resume Parser that extracts candidate data using spaCy and PDF parsers with a GUI in Python.
 Project Features:
 Language: Python
 GUI: Tkinter
 Output Format: JSON + Excel
 Resume Format: PDF + DOCX

 Core Libraries:
 spaCy (NER)
 re (Regex)
 pandas, openpyxl, json
 docx2txt, pdfplumber for file reading
